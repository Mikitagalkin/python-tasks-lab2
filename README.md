# Лабораторная работа №2: Структурное программирование на Python

## Цель работы
Освоить базовые конструкции Python: переменные, условия, циклы, функции, PEP8.

## Теоретические положения
В работе используются следующие конструкции Python:
- Переменные и типы данных: int, float, str, list, dict
- Условные операторы if/elif/else
- Циклы for, while
- Функции: def, параметры, return
- Стиль кода PEP8
- Модули math, random

## Описание заданий

### Задания средней сложности

#### 1. НОД двух чисел (task4_gcd.py)
**Демонстрирует:**
- Переменные и типы данных (int)
- Условные операторы if/elif/else
- Цикл while
- Функции с параметрами
- Модуль math

**Описание:** Программа находит наибольший общий делитель двух чисел с использованием алгоритма Евклида.

**Пример работы:**
Введите первое число: 48
Введите второе число: 36
НОД(48, 36) = 12

#### 2. Калькулятор (task6_calculator.py)
**Демонстрирует:**
- Переменные и типы данных (int, float)
- Условные операторы
- Циклы while
- Словари (dict) для хранения операций
- Обработка исключений

**Описание:** Программа выполняет основные арифметические операции.

**Пример работы:**
Введите операцию: +
Введите первое число: 10
Введите второе число: 5
10.0 + 5.0 = 15.0


#### 3. Словарь квадратов чисел (task10_squares_dict.py)
**Демонстрирует:**
- Тип данных dict
- Генераторы словарей
- Циклы for
- Методы работы со словарями

**Описание:** Программа создает словарь, где ключи - числа, значения - их квадраты.

**Пример работы:**
Введите N: 5
Словарь квадратов:
1 → 1
2 → 4
3 → 9
4 → 16
5 → 25


### Задания повышенной сложности

#### 4. Игра «Угадай число» (task5_guess_game.py)
**Демонстрирует:**
- Модуль random
- Сложные условные конструкции
- Вложенные циклы
- Работу со списками и словарями
- Обработку исключений

**Описание:** Интерактивная игра с несколькими уровнями сложности.

**Уровни сложности:**
- Легкий: 1-50, 10 попыток, с подсказками
- Средний: 1-100, 7 попыток, с подсказками
- Сложный: 1-200, 5 попыток, без подсказок
- Эксперт: 1-500, 3 попытки, без подсказок

#### 5. Рекурсивная сумма чисел (task9_recursive_sum.py)
**Демонстрирует:**
- Рекурсивные функции
- Базовый случай рекурсии
- Сравнение рекурсивного и итеративного подходов
- Модуль sys для управления глубиной рекурсии

**Описание:** Программа демонстрирует различные способы вычисления суммы с использованием рекурсии.
